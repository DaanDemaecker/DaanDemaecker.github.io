Description

During my Game Development course at Digital Arts and Entertainment in Belgium, we made our own basic 2D engine in our Programming 4 class. This inspired me to make my own basic 3D engine.
In my Graphics Programming class, we learned about DirectX and made our own software rasterizer, this made me look more into graphics programming and graphics API's. After some research, I decided to use the Vulkan API to render.
As I was new to Vulkan I started by creating a separate renderer that turned into a testing project, which can also be found on my GitHub.
